Unityでマルチプレイするとしたら何を使いますか？
Photon？ Monobit？ 自前でサーバー構築？
いやいやそこはUnity謹製のUNETでしょ！
ということで今回はUNETについて書いていきたいと思います
と言いつつも自分もこれから学びながらなので間違い等があったら指摘して頂けるとありがたいです

まず空のオブジェクトを生成してそれにNetworkManagerをアタッチします
今回はさらにNetworkManagerHUDもこのオブジェクトにアタッチしています
そのあとにPlayerオブジェクトを生成します（今回はCubeにしました）
このPlayerオブジェクトにはNetwokrIdentityをアタッチします
アタッチしたらNetworkIdentityのLocal Player Authorityにチェックをつけます
このPlayerオブジェクトをNetworkManagerのSpawn InfoのPlayer Prefabにアタッチします

スクショを二つ春

この状態でビルドして実行したのちにエディタで実行をして、一つをHostとしてもう一つをClientとするとゲーム上に二つのCubeがいることになります（ポジションが同じなので画面上では一つに見えますが…）
スクショを貼る

これでネットワークの基礎部分は出来たことになります！
なんて簡単なんでしょう！！

これから動きの部分を作っていきます
先ほど作ったPlayerオブジェクトに適当に新しいスクリプトをアタッチして下のコードを書きます

using UnityEngine;
using UnityEngine.Networking;

public class PlayerMove : NetworkBehaviour
{
	void Update()
	{
		if(!isLocalPlayer) return;

		var x = Input.GetAxis("Horizontal")*0.1f;
		var z = Input.GetAxis("Vertical")*0.1f;

		transform.Translate(x, 0, z);
	}
}

Updateの中のifの条件はまんまローカルプレイヤーだったらという条件です
こちらのifがない状態で先ほどと同じように二つ実行してみると二つ表示されているCubeが同じ動きをしてしまいます
